Creating a Repository

If you don’t have an existing repository you would like to use, you can create a new repository with PolyGit.

Initializing an Empty Repository

  1. Go to the Repositories screen.
  2. Select More.
  3. Select New Repository.
  4. Enter the name you would like to give your repository.
  5. Select Create.
  6. A new repository will be created by PolyGit.
