Keyboard Shortcuts

PolyGit supports a wide variety of keyboard shortcuts when connected to an external keyboard. You can view this list in the app’s settings or by holding down the ⌘ key.

General

Name Shortcut
Go Forward ⌘ Ctrl ]
Go Backward ⌘ Ctrl [
New File ⌘ N
New Folder ⌘ Shift N
Import File ⌘ O
Quick Open ⌘ Shift O
Preview ⌘ R
Preview in Safari ⌘ Shift R
Toggle Sidebar ⌘ 0
Switch to Files ⌘ 1
Switch to Status ⌘ 2
Switch to Branches ⌘ 3
Switch to Commits ⌘ 4
Switch to Search ⌘ 5

Text Editing

Name Shortcut
Indent ⌘ ]
Unindent ⌘ [
Move Line Up ⌘ Ctrl ↑
Move Line Down ⌘ Ctrl ↓
Select Word ⌘ D
Select Line ⌘ L
Select All ⌘ A
Find ⌘ F
Find with Selection ⌘ E
Find Next ⌘ G
Find Previous ⌘ Shift G
Increase Font Size ⌘ +
Decrease Font Size ⌘ -
